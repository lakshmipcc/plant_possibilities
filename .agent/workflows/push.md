---
description: How to push changes to GitHub
---

Follow these steps to push your recent changes (logic reversal and database expansion) to GitHub:

1. **Stage your changes**:
   ```bash
   git add .
   ```

2. **Commit your changes**:
   ```bash
   git commit -m "Reverse identification logic and expand plant database"
   ```

// turbo
3. **Push to GitHub**:
   ```bash
   git push origin main
   ```

> [!NOTE]
> If you haven't set up your Personal Access Token (PAT) yet, you may be prompted for your username and password. Use your PAT as the password.
